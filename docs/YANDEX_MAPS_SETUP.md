# Настройка Яндекс Карт

## Получение API ключа

1. Перейдите на [Яндекс Developer Dashboard](https://developer.tech.yandex.ru/)
2. Войдите или зарегистрируйтесь
3. Нажмите "Получить ключ"
4. Выберите сервис "JavaScript API и HTTP Геокодер"
5. Заполните форму с контактной информацией
6. Скопируйте полученный ключ

## Настройка проекта

1. Откройте файл `.env` в корне проекта
2. Замените `YOUR_API_KEY_HERE` на ваш API ключ:
   ```
   VITE_YANDEX_MAPS_API_KEY=ваш_ключ_здесь
   ```

3. Также обновите ключ в файле `src/components/YandexMap.tsx` на строке 38:
   ```typescript
   script.src = 'https://api-maps.yandex.ru/v3/?apikey=ваш_ключ_здесь&lang=ru_RU';
   ```

```bash

npm run dev
```

## Возможности карты

- Интерактивная карта Москвы
- Маркеры объектов недвижимости с ценами
- Маркеры покупателей
- Режим отображения ликвидности (зеленый/желтый/красный)
- Клик по маркеру открывает паспорт объекта
- Hover эффекты на маркерах
