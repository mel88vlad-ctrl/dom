# Настройка API ключа Яндекс Карт

## Проблема

Ошибка загрузки Яндекс Карт обычно связана с тем, что для API ключа не настроены ограничения по HTTP Referer.

## Решение

### Шаг 1: Откройте настройки ключа

1. Перейдите на https://developer.tech.yandex.ru/
2. Войдите в аккаунт
3. Найдите раздел "Ключи" или "Keys"
4. Найдите ваш ключ для JavaScript API

### Шаг 2: Настройте ограничения по HTTP Referer

В настройках ключа найдите поле "Ограничение по HTTP Referer" и добавьте:

```
http://localhost:3000/*
http://localhost:*
http://127.0.0.1:*
```

Или для разработки можно временно использовать:
```
*
```
(разрешает все домены, но не рекомендуется для продакшена)

### Шаг 3: Сохраните изменения

После сохранения изменения вступают в силу в течение нескольких минут.

### Шаг 4: Перезагрузите страницу

Обновите страницу в браузере (F5 или Ctrl+R).

## Альтернативное решение: Использовать API v2.1

Если проблемы с v3 продолжаются, можно временно использовать старую версию API:

### Обновите YandexMap.tsx:

Замените URL загрузки скрипта:
```typescript
// Вместо v3
script.src = `https://api-maps.yandex.ru/v3/?apikey=${apiKey}&lang=ru_RU`;

// Используйте v2.1
script.src = `https://api-maps.yandex.ru/2.1/?apikey=${apiKey}&lang=ru_RU`;
```

## Проверка ключа

Откройте консоль браузера (F12) и проверьте:
1. Какой ключ используется (первые 8 символов)
2. Полный URL скрипта
3. Детали ошибки

## Время активации

⏰ Новые ключи активируются в течение 15 минут после создания.

Если ключ создан только что, подождите 15 минут и попробуйте снова.

## Проверка в браузере

Откройте в новой вкладке:
```
https://api-maps.yandex.ru/v3/?apikey=ваш_ключ&lang=ru_RU
```

Если ключ работает, вы увидите JavaScript код, а не ошибку.
