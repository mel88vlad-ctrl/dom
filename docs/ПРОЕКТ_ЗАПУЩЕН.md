## Важно! Настройка Яндекс Карт

⚠️ Для работы карты нужен API ключ Яндекс Карт:

### Быстрая настройка (2 минуты):

1. Перейдите: https://developer.tech.yandex.ru/
2. Войдите/зарегистрируйтесь
3. Нажмите "Получить ключ"
4. Выберите "JavaScript API и HTTP Геокодер"
5. Заполните форму
6. Скопируйте ключ

7. Откройте файл `.env` в корне проекта
8. Замените `YOUR_API_KEY_HERE` на ваш ключ:
   ```
   VITE_YANDEX_MAPS_API_KEY=ваш_ключ_здесь
   ```

9. Перезапустите frontend (Ctrl+C в терминале frontend, затем `npm run dev`)

## API Endpoints

Backend API доступен на http://localhost:3001/api

- `POST /api/auth/register` - Регистрация
- `POST /api/auth/login` - Вход
- `GET /api/auth/me` - Текущий пользователь
- `GET /api/properties/search` - Поиск объектов
- `GET /api/properties/:id` - Детали объекта
- `POST /api/offers` - Создать оффер
- `GET /api/offers/property/:propertyId` - Офферы по объекту

## Поддержка

Если возникли проблемы:
1. Проверьте, что PostgreSQL запущен
2. Проверьте, что база данных `truedom` создана
3. Проверьте, что seed данные загружены (`npm run db:seed` в папке backend)
4. Проверьте консоль браузера на ошибки
5. Проверьте логи backend в терминале
