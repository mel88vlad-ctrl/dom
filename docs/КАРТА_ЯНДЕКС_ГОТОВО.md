## Быстрый старт

### Шаг 1: Получите API ключ Яндекс Карт

1. Перейдите на https://developer.tech.yandex.ru/
2. Войдите или зарегистрируйтесь
3. Нажмите "Получить ключ"
4. Выберите "JavaScript API и HTTP Геокодер"
5. Заполните форму
6. Скопируйте ключ

### Шаг 2: Настройте .env файл

Откройте `.env` в корне проекта и добавьте ваш ключ:

```env
VITE_YANDEX_MAPS_API_KEY=ваш_ключ_здесь
```

### Шаг 3: Запустите приложение

```bash
# Backend (если еще не запущен)
cd backend
npm run dev

# Frontend (в другом терминале)
npm run dev
```

## Примечания

- API ключ активируется в течение 15 минут после создания
- Для работы карты нужно заполнить поле "Ограничение по HTTP Referer" в настройках ключа
- Карта загружается асинхронно, показывается индикатор загрузки
- Все координаты проверены и соответствуют реальным адресам в Москве
