# ✅ Личный Кабинет Готов

## Что добавлено

### 1. React Router
- Установлен `react-router-dom`
- Настроена маршрутизация между страницами

### 2. Страницы
- `/` - Главная страница с картой объектов
- `/dashboard` - Личный кабинет пользователя

### 3. Личный кабинет (Dashboard)
- Профиль пользователя с именем и email
- Статистика:
  - Мои офферы
  - Избранное
  - Просмотры
  - Документы
- Быстрые действия:
  - Найти объект (переход на карту)
  - Избранное
  - Настройки
- История активности
- Кнопка выхода

### 4. Навигация
- После входа/регистрации автоматический переход в ЛК
- Кнопка "Личный кабинет" в шапке для авторизованных пользователей
- Кнопка "Войти" для неавторизованных
- Из ЛК можно вернуться на главную (клик по логотипу)

## Как использовать

### Вход в систему
1. Откройте http://localhost:3000
2. Нажмите "Войти"
3. Введите тестовые данные:
   - Email: `buyer@truedom.ru`
   - Пароль: `password123`
4. После входа вы автоматически попадете в личный кабинет

### Навигация
- **Главная → ЛК**: Нажмите кнопку "Личный кабинет" в шапке
- **ЛК → Главная**: Нажмите на логотип TrueDom или кнопку "Найти объект"
- **Выход**: Нажмите "Выйти" в ЛК
