# ✅ Админ-панель готова!

## Что добавлено

### Админ-панель (`/admin`)
- Боковое меню с навигацией
- Обзор системы с статистикой:
  - Количество пользователей
  - Количество объектов
  - Количество офферов
  - Общий оборот
- Разделы (в разработке):
  - Управление пользователями
  - Управление объектами
  - Управление офферами
- Быстрые действия
- Профиль администратора

### Доступ
- Только для пользователей с ролью `agent` или `admin`
- Автоматическая проверка прав доступа
- Редирект на главную если нет прав

### Навигация
- Кнопка "Админ-панель" в личном кабинете (только для агентов)
- Прямой доступ по URL: `/admin`

## Как использовать

### Вход в админку

1. Войдите как агент:
   - Email: `agent@truedom.ru`
   - Пароль: `password123`

2. В личном кабинете нажмите кнопку "Админ-панель"
   
   ИЛИ
   
   Перейдите напрямую: http://localhost:3000/admin

### Функционал

✅ **Обзор системы**
- Статистика по пользователям, объектам, офферам
- Быстрые действия
- Переход на карту

🚧 **В разработке**
- Управление пользователями
- Управление объектами
- Управление офферами
- Экспорт данных
- Настройки системы

## Роли пользователей

- `buyer` - покупатель (нет доступа к админке)
- `seller` - продавец (нет доступа к админке)
- `agent` - агент (есть доступ к админке)
- `admin` - администратор (есть доступ к админке)

## Тестовые аккаунты

```
Агент (доступ к админке):
  Email: agent@truedom.ru
  Пароль: password123

  Email: buyer@truedom.ru
  Пароль: password123

Продавец (нет доступа):
  Email: seller@truedom.ru
  Пароль: password123
```
