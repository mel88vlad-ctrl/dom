## 🔌 API интеграция (services/api.ts)

### Реализованные модули:

#### 1. **Управление токенами** ✅
```typescript
- setAuthToken() - сохранение токена
- getAuthToken() - получение токена
- clearAuthToken() - удаление токена
- localStorage интеграция
```

#### 2. **Auth API** ✅
```typescript
- register() - регистрация пользователя
- login() - вход
- getCurrentUser() - получение текущего пользователя
- updateProfile() - обновление профиля
- logout() - выход
```

#### 3. **Properties API** ✅
```typescript
- search() - поиск объектов с фильтрами
- getById() - получение по ID
- getByCadastral() - получение по кадастровому номеру
- create() - создание объекта
- update() - обновление
- delete() - удаление
```

**Поддерживаемые фильтры:**
- Город, районы
- Диапазон цен
- Количество комнат
- Площадь (мин/макс)
- Этаж (мин/макс)
- Статус ремонта
- Тип здания
- Сортировка (цена, площадь, дата, ликвидность)

#### 4. **Listings API** ✅
```typescript
- create() - создание листинга
- getById() - получение по ID
- getByProperty() - листинги объекта
- getMy() - мои листинги
- update() - обновление
- markAsSold() - пометить как проданный
- withdraw() - снять с продажи
- delete() - удаление
```

#### 5. **Offers API** ✅
```typescript
- create() - создание оффера
- getByProperty() - офферы объекта (Order Book)
- getMy() - мои офферы
- getById() - получение по ID
- accept() - принять оффер (владелец)
- reject() - отклонить оффер
- withdraw() - отозвать оффер (покупатель)
```

### Особенности API клиента:
- ✅ Централизованная обработка ошибок
- ✅ Автоматическое добавление Authorization header
- ✅ TypeScript типизация всех запросов
