## 🚀 Как Запустить

### Вариант 1: Быстрый Старт (Рекомендуется)

```powershell
# 1. Запустите Backend
cd backend
npm run dev

# 2. В новом терминале: Заполните базу данных
cd backend
npm run db:seed

# 3. В новом терминале: Запустите Frontend
npm run dev
```

### Вариант 2: С Нуля

```powershell
# 1. Создайте базу данных
createdb truedom

# 2. Инициализируйте схему
cd backend
npm run db:init

# 3. Заполните тестовыми данными
npm run db:seed

# 4. Запустите Backend
npm run dev

# 5. В новом терминале: Запустите Frontend
cd ..
npm run dev
```
